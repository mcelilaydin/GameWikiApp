# GameWikiAPP
<p align="center">
    Game Wiki application allows you to get information about games. You can find the game you want as you wish and get information about the game. You can add your favorite games to favourites. Have fun 😊🎉
    </p>
    
## Features 🔥

- Onboarding
- MVC Design Pattern
- Tabbar
- UICollectionView
- Custom CollectionView Cell
- Pinterest Layout
- UITableView
- Custom TableView Cell
- Core Data with Manager
- Notification Center
- SDWebImage
- Combine
- MBProgressHUD
- Loaf
- SCLAlertView
- PaperOnboarding

## Requirements 🛠️

- iOS 9.0+
- Xcode 16.2
